### 本地存储

应用使用 `UserDefaults` 和 `SwiftData` 在设备本地存储以下信息：

| 存储项 | 用途 | 类型 |
|--------|------|------|
| 音效音量 | 保存玩家音效偏好设置 | 浮点数 |
| 教程完成标记 | 记录玩家是否完成新手教程 | 布尔值 |
| 购买状态 | 记录应用内购买解锁状态 | 布尔值 |
| 游戏进度存档 | 保存当前冒险进度（关卡、分数、手牌等） | 数据库 |

以上数据完全存储于用户设备本地，不会上传至任何服务器。

## English Version

### App Information

- **App Name**: 斗破乾坤 (Dou Po Qian Kun)
- **Bundle ID**: com.hongzeng.doudizhu
- **Developer**: Individual developer

### Data Collection Statement

**This app does not collect any personal information.**

Dou Po Qian Kun is a purely offline single-player card game. It contains no network requests, no server communication, no third-party analytics SDKs, and no advertising frameworks.

### Local Storage

The app uses `UserDefaults` and `SwiftData` to store the following data locally on the device:

| Item | Purpose | Type |
|------|---------|------|
| Sound volume | Save player sound preferences | Float |
| Tutorial completion flag | Track whether player has completed the tutorial | Boolean |
| Purchase status | Track in-app purchase unlock state | Boolean |
| Game save data | Save current adventure progress (floor, score, hand, etc.) | Database |

All data is stored entirely on the user's device and is never uploaded to any server.

### Third-Party Services

- ❌ No third-party analytics SDKs
- ❌ No advertising frameworks
- ❌ No social login
- ❌ No cloud storage or server communication
- ✅ Apple Game Center (optional): Used for leaderboards and achievements, managed by Apple

### Children's Privacy

This app does not collect any information from children under 13. Since the app collects no personal data whatsoever, it complies with COPPA requirements.

### No Personally Identifiable Information

This application does not collect, store, transmit, or share any personally identifiable information (PII). No names, email addresses, phone numbers, location data, device identifiers, or any other PII is accessed by this application.
